  1. Context-Free Grammars

    In this question, I ask you to define context-free grammars as constants using let with a fixed name given below in each question. There is already a placeholder in homework7.ml for those answers. Just replace the placeholder with your definition.

    1. Define a context-free grammar palindromes that generates all palindromes over alphabet {a,b,c}. Remember that a palindrome is a string that reads the same left-to-right and right-to-left, such as aba or ccddcc.

    2. Define a context-free grammar ambncmn that generates the language {ambncm+n | m, n ≥ 0}, i.e., the set of all strings made up of as followed by bs followed by cs, where there are as many cs as as and bs together.

    3. Define a context-free grammar amcmnbn that generates the language {amcm+nbn | m, n ≥ 0}, i.e., the set of all strings made up of as followed by cs followed by bs, where there are as many cs as as and bs together.

    4. Define a context-free grammar ambncm that generates the language {ambncm | m, n ≥ 0}, i.e., the set of all strings made up of as followed by bs followed by cs, where there are as many as as cs.

    5. Define a context-free grammar eqnum that generates all strings over {d,e} that have an equal number of ds and es.
