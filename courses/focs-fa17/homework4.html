  1. Higher-Order Functions (1)

    For each function you have to code in this question, full points will be awarded if the function does not use explicit recursion — that is, if you use List.map or List.filter or List.fold_right. If you can't get it to work without recursion, please provide a version with explicit recursion.

    1. Code a function nfilter of type ('a -> bool) -> 'a list -> 'a list where nfilter p xs returns the list of elements in xs that do not satisfy predicate p (i.e., for which p is false)

      Show sample output

      # nfilter (fun x -> x > 0) [];;
- : int list = []
# nfilter (fun x -> x > 0) [1];;
- : int list = []
# nfilter (fun x -> x > 0) [1;-2];;
- : int list = [-2]
# nfilter (fun x -> x > 0) [1;-2;3;-4;5];;
- : int list = [-2; -4]
# nfilter (fun x -> x = "") [];;
- : string list = []
# nfilter (fun x -> x = "") ["";"";"";""];;
- : string list = []
# nfilter (fun x -> x = "") ["a";"";"b";"";"c"];;
- : string list = ["a"; "b"; "c"]

  2. Higher-Order Functions (2)

    For each function you have to code in this question, full points will be awarded if the function does not use explicit recursion — that is, if you use List.map or List.filter or List.fold_right. If you can't get it to work without recursion, please provide a version with explicit recursion.

    1. Code a function prepend of type 'a -> 'a list list -> 'a list list where preprend x xss returns the list obtained by prepending x to every list in xss.

      Show sample output

      # prepend 1 [];;
- : int list list = []
# prepend 1 [[]];;
- : int list list = [[1]]
# prepend 1 [[66]];;
- : int list list = [[1; 66]]
# prepend 1 [[66];[99]];;
- : int list list = [[1; 66]; [1; 99]]
# prepend 1 [[66];[99];[]];;
- : int list list = [[1; 66]; [1; 99]; [1]]
# prepend 1 [[66];[99];[];[66;99]];;
- : int list list = [[1; 66]; [1; 99]; [1]; [1; 66; 99]]
# prepend "+" [["hello";"world"];["goodbye"]];;
- : string list list = [["+"; "hello"; "world"]; ["+"; "goodbye"]]

    2. Code a function prefixes of type 'a list -> 'a list list where prefixes xs returns the list of all prefixes of xs: if xs is [x1; x2; x3] then the prefixes of xs are [], [x1], [x1; x2], and [x1; x2; x3]. (Note that the empty list is a prefix of every list.)
